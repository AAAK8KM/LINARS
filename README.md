#LINARS

To build use
```
mkdir build
cmake -S . -B build
cmake --build build
```

To test use
```
cmake --build mtest
```
or in build dir
```
ctest
```

# Некоторые замечания

1) Умножение матрицы на вектор сделаны с помощью итератора, поэтому оно одинаково и поисано в `imatrix.hpp` после класса `IMatrix<dtype>`

2) В итераторах используются сырые указатили и (о боже) static_cast на потомка, но это безопасно, уверяю. Пока пользователь гарантирует существование экземпляра итератор на него удет валиден. Единственое но - пользвоатель сам должен следить, чтобы не выйти за границы массива

3) Плотная матрица - `Matrix<dtype>`

4) Для запуска всех тестов есть цель `mtest`

5) Измерения в measures
